Layout

The layout of this kitchen sink page itself uses CSS Flexbox (see `kitchen-sink.page.scss`) to arrange component sections and items within rows. Material Design emphasizes responsive grids (often 4, 8, or 12 columns) and consistent spacing (typically 8dp increments).

Components like Cards and Inputs here are set to be flexible within their rows.
